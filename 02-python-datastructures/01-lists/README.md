# 📝 Python Lists - Notes

Lists in Python are ordered, mutable collections that can store elements of different data types. This guide includes syntax, methods, and practical tasks.

---

## 📌 Creating Lists

```python
fruits = ["apple", "banana", "cherry"]
numbers = [1, 2, 3, 4, 5]
mixed = [1, "hello", 3.5, True]

print(fruits)
print(numbers)
print(mixed)
```
